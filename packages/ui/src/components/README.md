# UI Components

These are shadcn/ui components, added on-demand.

## Adding a component

```bash
cd packages/ui
pnpm dlx shadcn@latest add button card input label
```

## Convention

- Components live in `src/components/<name>.tsx`
- Helpers in `src/lib/`
- Hooks in `src/hooks/`
- All exported via package.json exports
