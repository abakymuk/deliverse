# Claude Code Skills

Reusable patterns for development sessions. Reference from prompts:

```
"Apply the feature-scaffold skill from docs/skills/feature-scaffold.md to create the X feature."
```

## Available skills

- [premortem.md](./premortem.md) — Before major work, imagine it failed
- [feature-scaffold.md](./feature-scaffold.md) — Create a new feature folder
- [code-review-self.md](./code-review-self.md) — Pre-commit self-check

## Adding new skills

When you do something more than twice, write a skill for it. Compound interest.

Naming: `<verb-noun>.md` (e.g., `add-shadcn-component.md`, `fix-failing-test.md`).
